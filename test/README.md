First, [build `libmatoya`](https://github.com/matoya/libmatoya/wiki/Building). The following compiler commands assume that your working directory is this test directory.

#### Windows
```
nmake
```

#### Linux / macOS
```
make
````

#### Coverage

- File
- JSON
- Log
- Memory
- Net
- Struct
- System
- TLS (via Net)
- Thread

- Version
